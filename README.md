# Royatul Aliyah Web App

Project repository for Royatul Aliyah Web App (https://royatulaliyah.netlify.app).

## Installation

Make sure nodejs is installed on your computer.

```sh
npm install
```

## Usage

```sh
npm run start
```

## Test

```sh
npm run test
```
